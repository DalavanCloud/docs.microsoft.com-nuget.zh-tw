# <a name="nuget-error-nu5031"></a>NuGet 錯誤 NU5031
<pre>The license file 'LICENSE.txt' has an invalid extension. Valid options are .txt, .md or none.</pre>

### <a name="issue"></a>問題

允許只列出的副檔名。

### <a name="solution"></a>方案

使用檔案的副檔名從允許清單。
