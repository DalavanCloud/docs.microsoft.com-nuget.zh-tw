# <a name="nuget-error-nu5020"></a>NuGet 錯誤 NU5020
<pre>A source file was added with an empty path.</pre>

### <a name="issue"></a>問題

原始程式檔已傳遞給`msbuild /t:pack`具備 empty path 的作業。


### <a name="solution"></a>方案

請確定所有來源檔案傳遞至`msbuild /t:pack`作業有有效的路徑，且存在於磁碟上。
