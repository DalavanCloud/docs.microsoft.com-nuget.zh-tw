# <a name="nuget-error-nu1108-previously-nu1606"></a>NuGet 錯誤 NU1108 (先前 NU1606)

<pre>Cycle detected: A -> B -> A</pre>

### <a name="issue"></a>問題
偵測到循環相依性。

### <a name="solution"></a>方案
封裝是在撰寫不正確;請連絡套件擁有者，以修正 bug。
