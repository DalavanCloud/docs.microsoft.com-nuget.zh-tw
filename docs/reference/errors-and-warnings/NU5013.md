# <a name="nuget-error-nu5013"></a>NuGet 錯誤 NU5013
<pre>Failed to build 'project.csproj'</pre>

### <a name="issue"></a>問題

專案無法建置時執行 NuGet pack 作業`-Build`選項。


### <a name="solution"></a>方案

請修正組建錯誤專案中，然後再試一次。 失敗的原因應該顯示在主控台記錄檔中。
