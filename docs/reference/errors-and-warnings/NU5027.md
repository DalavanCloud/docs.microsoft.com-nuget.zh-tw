# <a name="nuget-error-nu5027"></a>NuGet 錯誤 NU5027
<pre>Invalid target framework for the file 'F:\project\project.dll'.</pre>

### <a name="issue"></a>問題

封裝組件不包含有效的目標 framework 中繼資料。


### <a name="solution"></a>方案

請確定 封裝組件包含有效`TargetFramework`屬性。
