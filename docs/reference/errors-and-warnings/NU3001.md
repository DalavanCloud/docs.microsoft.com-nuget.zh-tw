# <a name="nuget-error-nu3001"></a>NuGet 錯誤 NU3001

*NuGet 4.6.0+*

### <a name="issue"></a>問題
無效的引數設為[登入命令](../../tools/cli-ref-sign.md)或[確認命令](../../tools/cli-ref-verify.md)。

### <a name="solution"></a>方案
檢查並修正提供的引數。
