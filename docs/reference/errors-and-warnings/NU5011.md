# <a name="nuget-error-nu5011"></a>NuGet 錯誤 NU5011
<pre>Unable to extract metadata from 'project.csproj'.</pre>

### <a name="issue"></a>問題

從組件或專案檔擷取中繼資料時發生執行階段例外狀況。


### <a name="solution"></a>方案

請建立問題 https://github.com/NuGet/Home/issues 。
