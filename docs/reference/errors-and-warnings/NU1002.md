# <a name="nuget-error-nu1002"></a>NuGet 錯誤 NU1002

<pre>'CLEAR' cannot be used in conjunction with other values</pre>

### <a name="issue"></a>問題
輸入與清除關鍵字的組合無效。

### <a name="solution"></a>方案
清除單獨使用，並略過所有其他輸入。
